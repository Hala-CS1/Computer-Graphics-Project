# Computer Graphics Project

This is a group project for the Computer Graphics course.

The project was developed using C++ and the graphics.h library. It demonstrates both static and dynamic space scenes, including graphical objects and animation techniques.

The project includes:
- Spaceship Design
- Stars and Planets
- Moon Illustration
- Static Scene Rendering
- Dynamic Scene Animation

The project was developed using:
- C++
- graphics.h Library
- Computer Graphics Concepts
- Mathematical Functions for Animation
